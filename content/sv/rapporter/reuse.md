Rapporten visar om Diggs öppna kodförråd är registrerade i det REUSE API som används av REUSE-badgen.
Den visar också vald licensmetadata från GitHub och kodförrådens filer.

## Källa

Rapportdata hämtas från `diggsweden/digg-scorecard-monitor`.
"Registrerad" betyder att REUSE API har statusdata för kodförrådet.
