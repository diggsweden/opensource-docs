The report shows whether Digg's open repositories have a `CODEOWNERS` file and whether that file contains owner rules.
It also lists the owners parsed from the file so maintainership is visible directly in the report.

## Source

Report data is fetched from `diggsweden/digg-scorecard-monitor`.
The check accepts `CODEOWNERS`, `.github/CODEOWNERS` and `docs/CODEOWNERS`.
